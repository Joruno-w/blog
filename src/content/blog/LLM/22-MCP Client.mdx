🤔 Claude Desktop 是如何使用 MCP Server 的？

1. 查看当前配置了哪些 MCP Server
2. 连接这些 MCP Server，查看这些 MCP Server 上面提供了哪些工具
3. 用户在提问的时候，会将工具箱（大模型能够使用的工具）连同问题一起发给大模型
4. 当 LLM 认为当前的问题涉及到工具的时候，会在回复汇总返回一个调用工具的标识
