创建配置文件，配置以下信息：

1. 需要解释的问题
2. 使用的模型
3. 模型温度参数
4. 最大并发数
5. 超时时间

```js
// 统一管理可调参数，便于复用与测试
export const TOPICS = [
  '闭包',
  '原型链',
  '事件循环',
  '作用域链',
  '宏任务/微任务',
]
export const MODEL = 'llama3'
export const TEMPERATURE = 0.7
export const MAX_CONCURRENCY = 2
export const TIMEOUT_MS = 60_000
```
