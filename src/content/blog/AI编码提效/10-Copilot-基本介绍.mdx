Co：Code（代码）

pilot：飞行员

Copilot 全称是 Github Copilot，它是一个AI 编程助手，可以通过**插件**的形式集成到 VSCode、JetBrains IDE、Neovim 等开发环境中，利用 OpenAI 的大语言模型来为你 **自动补全代码、生成代码、解释代码**，甚至能根据注释写出整段逻辑。

## 它能做什么

安装好 Copilot 之后，后面的操作基本上就和 Cursor 差不多，支持：

1. 代码自动补全
2. 根据注释写代码
3. 重构建议：根据上下文尝试重写、优化你已有的代码，比如更简洁地实现某些逻辑。
4. 测试生成：你写好一个函数，Copilot 可以帮你自动生成相应的测试用例。

整个 Copilot 支持的编程语言也非常丰富，包括但不限于

- JavaScript / TypeScript
- Python
- Java
- C/C++
- Go
- Rust
- HTML/CSS
- SQL
- Bash 等
